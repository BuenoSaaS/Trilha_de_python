Aprender a linguagem


Desafio - Semana 1

O programa inicialmente irá captar os dados oferecidos pelo usuário referentes aos custos da viagem, orçamento disponível, local de destino e duração da viagem, nesse momento realizando verificações para identificar incoerências nos valores dados respectivos às informações, exigindo que sejam colocados novamente em caso positivo de erros. Em seguinte, calcula já realizando as conversões necessárias quanto seria necessário para hospedagem durante o período referente e para a viagem como um todo, verificando se o orçamento prévio é capaz de cobrir os valores finais e atribui-se situações de viabilidade ou não por meio dos operadores condicionais, também calculando quanto sobraria ao realizar ao viagem ou precisaria para que seja realizada. Por fim, realiza prints para exibir os resultados gerais ao usuário certificando aproximações decimais para os valores e uma condicional final para dizer se há uma sobra ou falta e qual valor de tal dado.
    Para utilizá-lo apenas insira os valores exigidos (Para o destino seria o nome da localidade) com base nas restrições exibidas (Não comporta valores negativos ou fora do estabelecido exemplo: Números em destino ou palavras em valores) e ao terminar o programa irá exibir os resultados diretamente ao usuário.

Perguntas Teóricas
1 - git add. irá atualizar todos os arquivos nos quais ocorreram mudanças enquanto git commit -m irá escrever uma mensagem para o commit em que será informada a função do commit, atualizar o conteúdo, corrigir erros, entre outros.
2 - A função input recebe os valores do usuário com o tipo string (str) e portanto não podem ser utilizados diretamente em operações aritméticas ou de matemática geral, sendo necessário transformá-los.
3 - Resultará em um erro visto que não é possível realizar operações com dados do tipo string, sendo necessário transformá-la em float.
